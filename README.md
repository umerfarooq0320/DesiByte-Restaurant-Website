# Desi Byte - Modern Restaurant Website

A sleek, contemporary, and fully mobile-responsive landing page for a premium South Asian fusion restaurant. This project focuses on delivering a seamless user experience, combining advanced CSS layouts and clean modular JavaScript to handle dynamic app-like structures without relying on heavy frameworks.

## Features

- **Interactive Navigation Sidebar:** A custom-engineered, slide-out drawer navigation menu tailored for a modern app-like mobile experience.
- **Glassmorphism Overlay:** Smooth, hardware-accelerated transitions that blur the background softly when the sidebar navigation is engaged.
- **Modern Bento Grid Menu:** An advanced CSS Grid template structure used to showcase culinary specialties in a beautiful, uneven masonry layout.
- **Fluid Mobile Responsiveness:** Comprehensive media queries that automatically scale and collapse multi-column desktop elements into a optimized, clean single-column vertical stack for smartphones.
- **High-DPI Branding Identity:** Designed with a geometric logo concept and sharp browser-rendered typography to prevent image pixelation on modern high-resolution screens.

- **HTML5:** Semantic and lightweight structural markup.

- **JavaScript (ES6+):** Pure event-driven DOM manipulation to manage navigation panel toggles.

1. Clone or download this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Toggle mobile view in your browser's developer tools to see the responsive layout adaptation.
